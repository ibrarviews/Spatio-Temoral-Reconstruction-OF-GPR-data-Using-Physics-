# Spatio-Temoral-Reconstruction-OF-GPR-data-Using-Physics-
Physics-guided spatio-temporal deep learning framework for time-lapse GPR reconstruction in karst systems. Integrates FDTD forward simulation, hybrid ConvLSTM-Transformer architecture, and physics-constrained training for subsurface permittivity reconstruction. Validated on Guilin karst field data. MATLAB implementation.
