📡 Physics-Guided GPR Data Reconstruction

📖 Overview

This repository presents a physics-guided framework for
Ground-Penetrating Radar (GPR) data reconstruction, combining domain
knowledge with computational techniques to enhance subsurface imaging
quality. The work focuses on improving reconstruction accuracy under
realistic field conditions by integrating physically meaningful
constraints into the data processing workflow.

The repository includes MATLAB implementations used to generate key
figures, along with access to field data used for validation.
